# Account Manager

Gerenciador de contas

# Subindo a aplicação

Execute o seguinte comando que esta na raiz do projeto

```sh
$ ./start.sh
```

ou rode manualmente

```sh
$ mvn clean install
$ docker-compose up -d
$ mvn spring-boot:run
```
